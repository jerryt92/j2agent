# 安全与用户

本目录归档 **j2agent** 与用户、认证、授权相关的实现说明。

## 文档索引

| 文档 | 说明 |
|------|------|
| [用户权限.md](./用户权限.md) | 用户表、角色、登录会话、`LoginInterceptor`、用户管理 REST API、内置 `aiadmin` |
| [邮箱注册机制.md](./邮箱注册机制.md) | 邮箱自助注册开关、白名单、验证码（Redis）、SMTP、公开注册 API |

## 路径约定

- **无需登录**：`/v1/auth/**`（登录、验证码、邮箱注册等）
- **需登录**：`/v1/rest/**`（具体接口角色见 [用户权限.md](./用户权限.md)）

配置类属性（含邮箱注册开关）存于表 `j2agent.ai_properties`，管理员可通过 Property API 维护。
